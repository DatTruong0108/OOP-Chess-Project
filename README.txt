1. Cài đặt thư viện SFML theo hướng dẫn trong đường link sau trước khi chạy: 
https://www.gamedevspot.net/index.php/forum/do-hoa/38-sfml-tutorial-ca-i-a-t-sfml-ep1-pha-n-1

2. Chương trình phải được build và thực thi ở hệ điều hành giống với hệ điều hành từ cài đặt SFML thì mới chạy được.
Ví dụ: Chương trình game Cờ Vua phải thực thi ở Solution Platform là x86 để phù hợp với phiên bản SFML đã cài đặt trước đó.

3. File thực thi .exe được đặt trong folder Debug ở cùng cấp với file .sln mới có thể chạy được chương trình.
